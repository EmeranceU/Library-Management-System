📚 Library Management System (Java OOP Project)
📌 Overview

This project is a simple Library Management System developed using Java. It demonstrates the core principles of Object-Oriented Programming (OOP) by modeling real-world library operations such as managing books, members, and borrowing activities.

🎯 Objective

To design a system that:

Manages books and members
Handles borrowing and returning of books
Demonstrates OOP concepts such as encapsulation, inheritance, abstraction, and polymorphism
🧱 System Design
Main Classes:
Person (Abstract Class) – Base class for all people
Member – Represents a library user
Librarian – Represents the library manager
Book – Represents a book in the library
Library – Handles system operations
🔗 Relationships
A Member can borrow a Book
The Library manages Books and Members
💡 OOP Concepts Used
Encapsulation
Private variables in classes (e.g., Book, Library)
Inheritance
Member and Librarian extend Person
Abstraction
Person is an abstract class
Polymorphism
displayInfo() method overridden in subclasses
⚙️ Features
Add books
Add members
Borrow books
Return books
Track book availability
🚀 How to Run

Compile all Java files:

javac *.java

Run the program:

java Main
